MODULE WEB-SUPPORT, 웹 계층 지원 모듈
===

# 개요

`:support:web` 모듈은 spring-web 의존성을 기반으로 web service를 제공하는 application에서 공통으로 사용되는 설정과 기능을 제공합니다.

# 구성

- [Logback](docs/description/logger/logback.md)
- BusinessException
- ExceptionHandler
- CommonErrorResponse
- ObjectMapper
- Utils
- Presentation Test Base
